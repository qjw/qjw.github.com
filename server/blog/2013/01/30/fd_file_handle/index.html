fd和FILE*相互转换

使用fileno函数从FILE指针获得fd。

使用fdopen函数打开一个fd并返回FILE指针

fd和HANDLE相互转换

使用_get_osfhandle函数从fd获得HANDLE

使用_open_osfhandle函数从HANDLE获得fd

重新打开

使用freopen函数重新打开FILE*

使用DuplicateHandle函数重新打开HANDLE

使用dup/dup2/dup3函数重新打开fd，但是不能变更权限，可以使用fcntl配合F_SETFL变更权限。
